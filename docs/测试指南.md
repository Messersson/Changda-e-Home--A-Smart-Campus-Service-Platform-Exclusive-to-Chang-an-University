# 测试指南

## 前端构建验证

```bash
cd frontend
npm run build
```

## Compose 配置验证

由于项目目录是中文名，建议显式传入 Compose 项目名：

```bash
docker compose -p chd-campus-dev -f docker-compose.dev.yml config
docker compose -p chd-campus-prod -f docker-compose.prod.yml config
```

## 后端构建验证

推荐通过 Docker 构建后端镜像：

```bash
docker compose -p chd-campus-dev -f docker-compose.dev.yml build backend
```

如果本机安装了 Maven，也可以执行：

```bash
cd backend
mvn test
```

## 手工接口验证

健康检查：

```bash
curl http://127.0.0.1:3000/api/health
```

登录管理员：

```bash
curl -X POST http://127.0.0.1:3000/api/auth/login ^
  -H "Content-Type: application/json" ^
  -d "{\"studentId\":\"2024000000\",\"password\":\"admin123\"}"
```

Swagger 文档：

```text
http://127.0.0.1:3000/swagger-ui.html
```
