# 本地部署说明

## 环境要求

- Docker Desktop
- Node.js 18+（仅用于运行启动脚本和前端本地构建）
- Java 17（仅当你不使用 Docker、希望本机运行后端时需要）
- Maven 3.9+（仅当你不使用 Docker、希望本机运行后端时需要）

## Docker Compose 部署

开发模式：

```bash
node scripts/start-platform.js
```

生产模式：

```bash
node scripts/start-platform.js --prod
```

停止服务：

```bash
node scripts/start-platform.js --stop
```

## 服务端口

开发模式：

- 前端：`http://127.0.0.1:5173`
- 后端：`http://127.0.0.1:3000`
- MySQL：`127.0.0.1:3306`

生产模式：

- 前端：`http://127.0.0.1:8080`
- 后端：`http://127.0.0.1:3000`

## 后端本机运行

如果本机已安装 Maven 和 MySQL，可以直接运行：

```bash
cd backend
mvn spring-boot:run
```

需要准备环境变量或修改 `backend/src/main/resources/application.yml` 中的数据库连接配置。

## 前端本机运行

```bash
cd frontend
npm install
npm run dev
```

Vite 开发服务器会把 `/api` 代理到 `http://localhost:3000`。
