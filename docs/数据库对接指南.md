# 长安大学校园服务平台 - 数据库对接指南

## 概述

本文档说明如何将长安大学校园服务平台从模拟数据切换到真实数据库。项目已预留所有数据库对接接口，只需实现 `DatabaseAdapter` 类中的方法即可完成对接。

## 数据库适配器架构

项目采用适配器模式，所有数据库操作都通过 `DatabaseAdapter` 类进行。当前实现为模拟数据，后续只需替换为真实数据库实现即可。

### 适配器位置
```
backend/database/adapter.js
```

### 适配器接口

```javascript
class DatabaseAdapter {
  // 数据库连接
  static async connect()
  
  // 数据库断开
  static async disconnect()
  
  // 执行自定义SQL查询
  static async query(sql, params)
  
  // 查询数据
  static async select(table, where, fields)
  
  // 插入数据
  static async insert(table, data)
  
  // 更新数据
  static async update(table, data, where)
  
  // 删除数据
  static async delete(table, where)
  
  // 执行事务
  static async transaction(callback)
}
```

### 第四步：实现数据库适配器

根据选择的数据库，实现 `DatabaseAdapter` 类。以下是各数据库的实现示例：

#### MySQL 实现示例

```javascript
const mysql = require('mysql2/promise')

class DatabaseAdapter {
  static connection = null

  static async connect() {
    this.connection = await mysql.createConnection({
      host: process.env.DB_HOST,
      port: process.env.DB_PORT,
      user: process.env.DB_USER,
      password: process.env.DB_PASSWORD,
      database: process.env.DB_NAME,
      multipleStatements: true
    })
    console.log('[数据库] MySQL 连接成功')
  }

  static async disconnect() {
    if (this.connection) {
      await this.connection.end()
      console.log('[数据库] MySQL 连接已断开')
    }
  }

  static async query(sql, params = []) {
    const [rows] = await this.connection.query(sql, params)
    return rows
  }

  static async select(table, where = {}, fields = '*') {
    const conditions = Object.keys(where).map(key => `${key} = ?`).join(' AND ')
    const values = Object.values(where)
    const sql = conditions 
      ? `SELECT ${fields} FROM ${table} WHERE ${conditions}`
      : `SELECT ${fields} FROM ${table}`
    return await this.query(sql, values)
  }

  static async insert(table, data) {
    const keys = Object.keys(data)
    const values = Object.values(data)
    const placeholders = keys.map(() => '?').join(', ')
    const sql = `INSERT INTO ${table} (${keys.join(', ')}) VALUES (${placeholders})`
    const [result] = await this.query(sql, values)
    return { insertId: result.insertId, affectedRows: result.affectedRows }
  }

  static async update(table, data, where) {
    const setClause = Object.keys(data).map(key => `${key} = ?`).join(', ')
    const whereClause = Object.keys(where).map(key => `${key} = ?`).join(' AND ')
    const values = [...Object.values(data), ...Object.values(where)]
    const sql = `UPDATE ${table} SET ${setClause} WHERE ${whereClause}`
    const [result] = await this.query(sql, values)
    return { affectedRows: result.affectedRows }
  }

  static async delete(table, where) {
    const whereClause = Object.keys(where).map(key => `${key} = ?`).join(' AND ')
    const values = Object.values(where)
    const sql = `DELETE FROM ${table} WHERE ${whereClause}`
    const [result] = await this.query(sql, values)
    return { affectedRows: result.affectedRows }
  }

  static async transaction(callback) {
    await this.connection.beginTransaction()
    try {
      const result = await callback()
      await this.connection.commit()
      return result
    } catch (error) {
      await this.connection.rollback()
      throw error
    }
  }
}

module.exports = DatabaseAdapter
```

#### PostgreSQL 实现示例

```javascript
const { Pool } = require('pg')

class DatabaseAdapter {
  static pool = null

  static async connect() {
    this.pool = new Pool({
      host: process.env.DB_HOST,
      port: process.env.DB_PORT,
      user: process.env.DB_USER,
      password: process.env.DB_PASSWORD,
      database: process.env.DB_NAME
    })
    await this.pool.connect()
    console.log('[数据库] PostgreSQL 连接成功')
  }

  static async disconnect() {
    if (this.pool) {
      await this.pool.end()
      console.log('[数据库] PostgreSQL 连接已断开')
    }
  }

  static async query(sql, params = []) {
    const client = await this.pool.connect()
    try {
      const result = await client.query(sql, params)
      return result.rows
    } finally {
      client.release()
    }
  }

  // 其他方法实现与 MySQL 类似
  // ...
}

module.exports = DatabaseAdapter
```

## 超市模块对接要点

### 1. 商品库存管理

超市下单时需要同时更新库存，必须使用事务保证数据一致性：

```javascript
await DatabaseAdapter.transaction(async () => {
  // 1. 扣减商品库存
  for (const item of cartItems) {
    await DatabaseAdapter.update('supermarket_products',
      { stock: product.stock - item.quantity },
      { id: item.productId }
    )
  }
  
  // 2. 创建订单
  await DatabaseAdapter.insert('orders', {
    user_id: userId,
    type: 'supermarket',
    items: JSON.stringify(cartItems),
    total_amount: totalAmount,
    address: address,
    phone: phone,
    status: 'pending'
  })
})
```

### 2. 购物车数据同步

购物车数据需要实时同步到数据库：

```javascript
// 添加到购物车
await DatabaseAdapter.insert('cart_items', {
  user_id: userId,
  product_id: productId,
  quantity: quantity
})

// 更新购物车
await DatabaseAdapter.update('cart_items',
  { quantity: newQuantity },

// 删除购物车商品
await DatabaseAdapter.delete('cart_items', {

  product_id: productId
```

## 注意事项

### 1. 数据类型映射

确保数据库字段类型与代码中的数据类型匹配：

| 代码类型 | MySQL 类型 | PostgreSQL 类型 |
|---------|------------|----------------|
| String | VARCHAR | VARCHAR |
| Number | INT/DECIMAL | INTEGER/NUMERIC |
| Boolean | TINYINT | BOOLEAN |
| Date | DATETIME | TIMESTAMP |
| JSON | TEXT/JSON | JSONB |

### 2. 字符编码

确保数据库使用 UTF-8 编码：

```sql
-- MySQL
CREATE TABLE ... ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

-- PostgreSQL
CREATE DATABASE ... ENCODING 'UTF8';
```

### 3. 索引优化

为常用查询字段添加索引：

```sql
-- 用户表索引
CREATE INDEX idx_student_id ON users(student_id);
CREATE INDEX idx_email ON users(email);

-- 商品表索引
CREATE INDEX idx_category_id ON supermarket_products(category_id);
CREATE INDEX idx_status ON supermarket_products(status);

-- 订单表索引
CREATE INDEX idx_user_id ON orders(user_id);
CREATE INDEX idx_type ON orders(type);
CREATE INDEX idx_status ON orders(status);
```

### 4. 事务使用

涉及多个表操作时必须使用事务：

```javascript
await DatabaseAdapter.transaction(async () => {
  // 所有数据库操作
  // 如果任何一个操作失败，整个事务会回滚
})
```

### 5. 错误处理

捕获并处理数据库错误：

```javascript
try {
  await DatabaseAdapter.insert('table', data)
} catch (error) {
  if (error.code === 'ER_DUP_ENTRY') {
    // 处理重复键错误
  } else {
    // 处理其他错误
  }
}
```

### 6. 连接池管理

使用连接池提高性能：

```javascript
// MySQL
const pool = mysql.createPool({
  connectionLimit: 10,
  host: process.env.DB_HOST,
  // ...
})

// PostgreSQL
const pool = new Pool({
  max: 10,
  host: process.env.DB_HOST,
  // ...
})
```


### 3. 死锁

**问题：** 事务死锁

**解决方案：**
