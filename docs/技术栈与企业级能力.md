# 技术栈与企业级能力

## 技术栈

- 后端：Java 17、Spring Boot 3、Spring MVC、Spring Security、Spring JDBC
- 前端：Vue 3、Vue Router、Pinia、Element Plus、Axios、Vite
- 数据库：MySQL 8、Flyway 数据库迁移
- 环境：Docker、Docker Compose、Nginx

说明：Vue 3 下对应的是 Element Plus，它是 Element UI 面向 Vue 3 的延续版本。

## 已加入的企业级能力

- 分层架构：Controller、Service、Repository 分离，业务逻辑从路由层下沉到服务层。
- 认证授权：JWT 无状态登录，Spring Security 保护接口，后台接口要求管理员角色。
- 数据迁移：Flyway 管理建表和初始化数据，环境启动时自动迁移。
- 统一响应：接口统一返回 `success/data/message/code`，兼容现有前端请求封装。
- 统一异常：全局异常处理器统一处理业务异常、认证异常、权限异常和参数异常。
- 事务控制：下单、购物车结算、支付确认、退款等关键流程使用事务保证一致性。
- 审计日志：通过 AOP 注解记录登录、发布、下单、支付、后台操作等关键行为。
- 运行监控：Spring Boot Actuator 暴露健康检查和指标端点。
- 接口文档：Springdoc OpenAPI 自动生成 Swagger 文档。
- 容器化部署：MySQL、后端、前端通过 Docker Compose 编排，前端 Nginx 反向代理 `/api`。

