# 平台启动说明

当前项目已改造为：

- 后端：Java 17 + Spring Boot 3
- 前端：Vue 3 + Element Plus（Element UI 的 Vue 3 版本）
- 数据库：MySQL 8
- 环境：Docker Compose

## 推荐启动方式

在项目根目录执行：

```bash
node scripts/start-platform.js
```

脚本会使用 `docker-compose.dev.yml` 启动 MySQL、Spring Boot 后端和 Vue/Nginx 前端，并自动等待服务健康检查完成。

默认访问地址：

- 用户端：`http://127.0.0.1:5173`
- 管理端：`http://127.0.0.1:5173/admin`
- 后端健康检查：`http://127.0.0.1:3000/api/health`
- Swagger 接口文档：`http://127.0.0.1:3000/swagger-ui.html`

## 生产模式

```bash
node scripts/start-platform.js --prod
```

生产模式使用 `docker-compose.prod.yml`，前端默认暴露在 `8080` 端口：

- 用户端：`http://127.0.0.1:8080`
- 管理端：`http://127.0.0.1:8080/admin`

## 停止服务

```bash
node scripts/start-platform.js --stop
```

生产模式停止：

```bash
node scripts/start-platform.js --prod --stop
```
