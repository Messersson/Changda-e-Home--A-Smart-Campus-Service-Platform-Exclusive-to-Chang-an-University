# 平台启动说明

## 推荐方式

在项目根目录直接双击 `启动平台.bat`。

启动器会自动完成以下事情：

- 检查 `backend` 和 `frontend` 是否存在
- 检查 `Node.js`、`npm` 和依赖是否已安装
- 自动启动后端服务（`3000` 端口）
- 自动启动前端服务（默认 `5173`，被占用时会顺延到 `5174+`）
- 等前后端都就绪后自动打开浏览器

## 命令行方式

如果你想在终端里启动，也可以执行：

```bash
node scripts/start-platform.js
```

如果只想做无弹窗验证，可执行：

```bash
node scripts/start-platform.js --headless --no-browser
```

## 常见问题

- 提示缺少依赖：先执行 `cd backend && npm install` 和 `cd frontend && npm install`
- 提示 `3000` 端口被占用：先关闭占用后端端口的程序再重试
- 前端不是 `5173`：这是 Vite 自动换端口，启动器会自动识别实际地址
