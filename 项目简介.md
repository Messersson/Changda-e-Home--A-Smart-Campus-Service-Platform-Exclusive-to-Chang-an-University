# 长安大学校园服务平台 - 项目简介

## 项目定位

长安大学校园服务平台是一套面向校园生活、学习资源、社区互动和后台运营的综合服务系统。平台把小吃外卖、校园超市、家教、二手交易、驾校咨询、学习资料、校园论坛和后台管理整合到统一入口，减少信息分散和管理成本。

## 当前技术栈

后端：

- Java 17
- Spring Boot 3
- Spring MVC
- Spring Security + JWT
- Spring JDBC
- Spring Boot Actuator
- Springdoc OpenAPI
- Flyway

前端：

- Vue 3
- Element Plus（Element UI 的 Vue 3 版本）
- Vue Router
- Pinia
- Axios
- Vite

数据库与环境：

- MySQL 8
- Docker
- Docker Compose
- Nginx

## 核心功能

用户端：

- 注册、登录、个人资料维护
- 东门小吃浏览、下单、订单取消
- 校园超市分类浏览、购物车、结算、库存扣减
- 家教信息浏览、发布、预约下单
- 二手商品浏览、发布、收藏、交易下单
- 驾校信息浏览、报名下单、咨询提交
- 学习资料浏览、上传、下载计数
- 校园论坛发帖、点赞、评论
- 模拟支付、支付确认、退款申请

后台端：

- 用户管理
- 小吃和超市商品管理
- 家教、二手、资料、论坛内容审核
- 订单状态管理
- 驾校和咨询管理
- 平台统计概览

## 企业级能力

项目已加入下列企业级开发实践：

- 分层架构：Controller、Service、Repository 职责拆分。
- 统一响应：接口统一返回 `success/data/message/code`。
- 统一异常：全局异常处理业务、认证、权限和参数错误。
- 安全认证：JWT 无状态认证，后台接口基于管理员角色授权。
- 数据迁移：Flyway 管理数据库建表和初始化数据。
- 事务控制：下单、购物车结算、支付、退款等流程使用事务。
- 审计日志：AOP 注解记录关键业务操作。
- 运行监控：Actuator 提供健康检查和基础指标。
- 接口文档：Swagger UI 自动生成接口文档。
- 容器编排：Docker Compose 编排 MySQL、后端和前端。

## 启动方式

推荐使用 Docker Compose 一键启动：

```bash
node scripts/start-platform.js
```

默认访问地址：

- 用户端：`http://127.0.0.1:5173`
- 管理端：`http://127.0.0.1:5173/admin`
- 后端健康检查：`http://127.0.0.1:3000/api/health`
- Swagger：`http://127.0.0.1:3000/swagger-ui.html`

生产模式：

```bash
node scripts/start-platform.js --prod
```

停止服务：

```bash
node scripts/start-platform.js --stop
```
